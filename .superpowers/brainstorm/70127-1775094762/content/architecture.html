Fluxo de Jobs Assíncronos

Solução para operações longas (extração de DS + geração de LP)

1. Usuário dispara ação
→
2. API cria job
status: pending
→
3. Processa
status: running
→
4. Conclui / Falha
status: done | error
→ Frontend faz polling a cada 3s consultando o status do job no Supabase · Sem WebSockets · Simples e confiável
