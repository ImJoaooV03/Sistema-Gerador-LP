Telas por Módulo

Referências
• Lista com filtros: niche · sub_niche · page_type · tags
• Card: thumbnail do index.html · nome · niche · tipo
• Ação: + Nova Referência → upload de ZIP
• Validação: alerta se faltar index.html ou design-system.html
• Detalhe: preview do index.html + design-system.html lado a lado
Design Systems
• Lista de DS cadastrados
• Ação: + Novo DS → upload de ZIP (index.html + assets)
• Status do job de extração em tempo real (polling)
• Preview do design-system.html gerado
• Botão: Download design-system.html
• Botão: Confirmar e salvar na biblioteca
Projetos
• Lista com status: rascunho · gerando · concluído
• Criar projeto: cliente · nome · niche · sub_niche · page_type · briefing
• Ação: Gerar LP → dispara job
• Histórico de versões geradas
Página Gerada (Editor)
• Preview: iframe (inline) + botão nova aba
• Lista lateral de seções/dobras detectadas
• Clique na seção → abre editor (código ou IA)
• Modo código: textarea com syntax highlight
• Modo IA: campo de prompt → reescreve seção
• Botão: Download HTML
• Botão: Gerar nova versão
Configurações
• API Keys por provedor: Claude · GPT-4o · outros
• Modelo padrão de IA (selecionável por projeto)
• Prompt de extração de DS (editável — base: "Extract HTML Design System v2")
• Prompt de geração de LP (editável)
